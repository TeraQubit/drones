##Run the application

```cmd
$mvn package
$mvn spring-boot:run
```

**http://localhost:8090/swagger-ui/index.html**

![alt text](https://github.com/TeraQubit/drones/blob/master/src/main/resources/static/documentation.png?raw=true "API Documentation")
